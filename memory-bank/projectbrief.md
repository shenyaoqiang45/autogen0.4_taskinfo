# Project Brief: 内容创作 Agent 系统

## 项目概述
基于 AutoGen 0.4+ 框架构建一个智能内容创作系统，用户只需提供主题，系统自动完善时间、地点、人物等细节，并提供可靠的引用来源。

## 核心目标
1. **主题扩展**: 接收用户提供的主题，自动生成完整的内容框架
2. **细节完善**: 智能补充时间、地点、人物等关键要素
3. **引用支持**: 为内容提供可靠的引用来源和参考资料
4. **多 Agent 协作**: 利用 AutoGen 0.4 的多 agent 能力实现专业化分工

## 主要功能
- 主题分析和理解
- 时间线构建（历史背景、关键时间点）
- 地点信息补充（地理位置、环境描述）
- 人物信息完善（相关人物、角色关系）
- 信息检索和引用（搜索可靠来源）
- 内容整合和输出（生成完整的创作内容）

## 技术栈
- AutoGen 0.4+
- Python 3.10+
- Tavily Search API（用于网络搜索）
- 高德地图 API（用于地点信息）

## 用户体验
用户输入一个主题 → 系统自动分析并完善细节 → 输出结构化的内容创作材料（包含时间、地点、人物、引用）

## 成功标准
- Agent 能够准确理解用户主题
- 自动补充的信息准确且相关
- 提供的引用来源可靠且可追溯
- 输出内容结构清晰、易于使用
